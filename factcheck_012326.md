## 3. 이론적 배경 - 팩트체킹 자동화

### 팩트체킹 자동화 3단계

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  1. 주장 탐지    │ ──▶ │  2. 증거 수집    │ ──▶ │  3. 사실 검증    │
│ Claim Detection │     │Evidence Retrieval│     │Verdict Prediction│
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

### 검증 가치(Check-worthiness)의 두 기준

| 기준 | 정의 | 예시 |
|------|------|------|
| **검증 가능성** | 통계, 공식 기록 등으로 입증 가능 여부 | "실업률이 5%에서 10%로 상승" |
| **검증 중요성** | 사회적 영향력, 정책적 판단에 미치는 영향 | 선거 공약, 정책 효과 주장 |

**예시 비교:**
- ❌ **가치판단:** "현 정부의 경제 정책은 실패했다"
- ✅ **검증 가능:** "현 정부 임기 중 실업률이 5%에서 10%로 상승했다"

### 한국어의 특수성

| 특성 | 설명 |
|------|------|
| **교착어** | 복잡한 어미 변화 (예: 먹+었+습니다) |
| **토큰화 어려움** | 띄어쓰기 규칙 복잡 |
| **존칭/경어** | 발화자 태도, 문장 확신 정도에 영향 |
| **데이터셋 부족** | 영어 데이터 전이 학습 필요 |

## 4. 이론적 배경 - LLM과 RAG

### LLM의 근본적 한계

| 한계 | 설명 |
|------|------|
| **지식 컷오프** | 학습 완료 시점 이후 정보 응답 불가 |
| **환각(Hallucination)** | 존재하지 않는 정보를 사실처럼 생성 |

### RAG(검색 증강 생성)의 해결 방식

```
사용자 질의 ──▶ 외부 소스 검색 ──▶ 관련 정보 수집 ──▶ LLM에 제공 ──▶ 증거 기반 응답
```

### 주요 선행 연구

| 연구 | 연도 | 특징 |
|------|------|------|
| **Ullrich et al. AVeriTeC** | 2024 | RAG 기반 팩트체킹, MMR 재순위화 |
| **Khaliq et al. RAGAR** | 2024 | Chain of RAG, Tree of RAG |
| **Li et al. FactAgent** | 2024 | 에이전트식 접근법, 단계별 검증 |
| **Zeng et al. RAGuard** | 2025 | 오도성 검색 취약성 지적 |

> ⚠️ **RAG 한계:** 잘못된 검색 결과에 과도하게 의존 가능 → 출처 신뢰도 평가 필요

## 7. 증거 수집 및 사실 판단 시스템

### 다중 소스 웹 검색

| 검색 소스 | 특징 | 수집 건수 |
|-----------|------|-----------|
| **DuckDuckGo** | 개인정보 보호 중심, 무료 | 최대 10건 |
| **한국어 위키피디아** | 기본 사실 정보, MediaWiki API | 최대 5건 |
| **Google (Serper.dev)** | 뉴스/학술 우선 | 최대 10건 |

### 출처 신뢰도 필터링

| 허용 도메인 | 차단 대상 |
|------------|----------|
| 정부기관 (go.kr, gov) | 신뢰도 낮은 개인 사이트 |
| 교육/연구기관 (edu, ac.kr) | 편향된 특정 도메인 |
| 국제기관, 주요 언론사 | - |

### LLM 기반 6단계 분류 체계

| 판단 | 정의 |
|------|------|
| **사실** | 충분한 긍정 근거, 부정 근거 없음 |
| **대체로 사실** | 긍정 근거 우세, 일부 부정 근거 |
| **사실 반 거짓 반** | 긍정/부정 근거 절반씩 |
| **대체로 거짓** | 부정 근거 우세, 일부 긍정 근거 |
| **거짓** | 논리적 근거 전혀 없음 |
| **판단 유보** | 자료 부족 또는 판정 불가 |

> **사용 LLM:** Google Gemini-2.5-Flash

```
                    ┌──────────────┐
                    │  입력 텍스트   │
```
