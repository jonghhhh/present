이론적 배경 - LLM과 RAG

LLM의 근본적 한계

한계설명
지식 컷오프학습 완료 시점 이후 정보 응답 불가
환각(Hallucination)존재하지 않는 정보를 사실처럼 생성

RAG의 해결 방식

질의 → 외부 검색 → 정보 수집 → LLM 제공 → 증거 기반 응답

논의 및 연구 함의

함의 1: 교차언어 전이 학습의 효과성

영어 데이터셋 번역 → 한국어 모델 개발 가능 → 데이터셋 부족 상황의 현실적 대안

함의 2: 인력 기반 팩트체킹 한계 보완

JTBC 56건 vs. 시스템 499건 → 약 9배 확대 → 선거 토론회 환경에서 효과적

함의 3: 혼합(Hybrid) 모델 제안

  • AI: 전체 발언 빠르게 분석 + 1차 선별
  • 인간: 복잡한 사안 최종 검토 + 맥락 이해
  • 결과: 속도 + 범위 + 품질 모두 향상

함의 4: 투명성·재현성 확보

모델 Hugging Face 공개 → 후속 연구 기반 마련 → 오픈 사이언스 기여
